Quick Setup Guide

  1. Login to Top.gg: Go to top.gg and log in via Discord.
  2. Find the Cookie: Open DevTools (F12) → Go to Application → Cookies → Select `https://top.gg`.
  3. Copy value: Locate the cookie named __Secure-authjs.session-token and copy its entire text value.
  4. Paste & Save: Paste it in the input field to the left, hit save, and you're fully configured!
